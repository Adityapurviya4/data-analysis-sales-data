# power bi 
Link to Sales Dataset:
https://www.kaggle.com/datasets/shreyanshverma27/online-sales-dataset-popular-marketplace-data

An interactive Power BI dashboard designed to analyze and visualize e-commerce sales performance. This project provides insights into key business metrics such as total sales, revenue trends, customer demographics, product performance, and regional sales distribution. It helps businesses track KPIs, identify growth opportunities, and make data-driven decisions.
